Hotel Management System

A cloud-based hotel management system built using Google Sheets, Google Forms, and Apps Script automation. It streamlines room booking, guest check-in/check-out, billing, and automated email confirmations via SMTP.

Features

  • Room booking system with availability tracking
  • Automated check-in/check-out process via Google Forms
  • Invoice & billing automation using Apps Script
  • Email confirmations sent via SMTP integration
  • Centralized MIS dashboard with guest history & occupancy reports

Screenshots

Technologies Used

Google Sheets, Google Forms, Google Apps Script, JavaScript, SMTP

Challenges & Learnings

The main challenge was managing real-time room availability in Google Sheets without conflicts. Learned advanced Apps Script techniques for triggers, form validation, and integrating Gmail SMTP for automated communication with guests.
