Bulk Email Sender

A VB.NET desktop application that allows users to send bulk emails with attachments, streamlining communication for businesses and HR teams.

Features

  • Import recipient list from Excel/CSV
  • Attach files to each email
  • Supports SMTP authentication
  • Delivery status logs

Screenshots

Technologies Used

VB.NET, Windows Forms, SMTP

Challenges & Learnings

Faced SMTP authentication issues with different email providers. Learned how to handle exceptions and optimize email batching for better performance.
